Preprocessor

SCSS variables & mixins

Tap into design tokens and utility mixins for scalable, maintainable Bangla typography systems.

Variables
SCSS 
@use 'bangla-ui-fonts/scss/variables' as *;

.my-text {
  font-family: $bangla-font-hind-siliguri;
}
Mixins
SCSS 
@use 'bangla-ui-fonts/scss' as *;

.text {
  @include bangla-body('hind-siliguri', 16px, 400);
}

.heading {
  @include bangla-heading('anek-bangla', 700);
}
Available Variables
SCSS 
// Font families
$bangla-font-hind-siliguri
$bangla-font-anek-bangla
$bangla-font-tiro-bangla
// ... and 13 more

// Font weights
$bangla-weight-regular    // 400
$bangla-weight-bold       // 700

Reference

Mixin API

Mixin Parameters Description
bangla-font() $font-name, $weight, $style Apply a font to any element
bangla-body() $font, $size, $weight Body text style preset
bangla-heading() $font, $weight Heading style preset

Framework

React components

Declarative components for Bangla text — typed, flexible, and tree-shakeable.

BanglaText
TSX 
import { BanglaText } from 'bangla-ui-fonts/react';

function App() {
  return (
    <BanglaText
      font="hind-siliguri"
      size="18px"
      weight={400}
    >
      আমার সোনার বাংলা
    </BanglaText>
  );
}
BanglaHeading
TSX 
import { BanglaHeading } from 'bangla-ui-fonts/react';

<BanglaHeading
  level={1}
  font="anek-bangla"
  weight={700}
>
  বাংলা শিরোনাম
</BanglaHeading>
BanglaFontProvider — Wrap your app
TSX 
import { BanglaFontProvider } from 'bangla-ui-fonts/react';

<BanglaFontProvider loadAll={true}>
  <YourApp />
</BanglaFontProvider>

API Reference

Component props

Prop Type Default Description
font BanglaFontName hind-siliguri Which typeface to apply
weight number 400 Font weight (e.g. 400, 700)
size string — Font size (e.g. "16px", "1.2rem")
level 1–6 1 Heading level (BanglaHeading only)

Vue 3 components

Plugin Setup
TypeScript 
import { createApp } from 'vue'
import App from './App.vue'
import { BanglaUIFonts } from 'bangla-ui-fonts/vue'

createApp(App)
  .use(BanglaUIFonts, { loadAll: true })

Types

Full TypeScript support

Bundled type definitions with strict font name unions and FontInfo interfaces — no DefinitelyTyped needed.

BanglaFontName Type
TypeScript 
import type { BanglaFontName } from 'bangla-ui-fonts'

const myFont: BanglaFontName = 'hind-siliguri'
// ✓ Type-safe — typos caught at compile time
FontInfo Type
TypeScript 
import { fonts, type FontInfo } from 'bangla-ui-fonts'

fonts.forEach((font: FontInfo) => {
  console.log(font.name)
})
